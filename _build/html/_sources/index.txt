Welcome to BrewLinux's documentation!
=====================================
BREWLinux is a debian based Linux system.
It is designed to built a basic system for lectures in computer sciences.

The base system consists in only a few tools.
The metapackages are more specific to lectures.

Download and install
--------------------
Find the `latest Version <vm/>`_ for BREWLinux

Development
-----------
If you found an issue, or you want to contribute to the project, we are on `github <https://github.com/BrewLinux>`_
